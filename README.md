# 2D Unity Physics Demo 

A 2d platformer demo in Unity using custom physics and collision handling.

## Features

 - Configurable acceleration and deceleration :runner:
 - Slopes can be traversed up to a certain steepness :mountain:
 - Variable jump height (configurable) :rabbit2:
 - One-way platforms :arrow_up_small:
 - Sample scene and prefabs :building_construction:
